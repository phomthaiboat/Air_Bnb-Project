# Asheville Airbnb Listings Analysis
โปรเจ็คนี้เป็นโปรเจ็ควิเคราะห์ Listings ของแอปพิเคชั่น AirBnb ในเมือง Ashvile โดยอาศัยข้อมูลจากเว็บไซต์ Analyst Builder ในการวิเคราะห์ 
โดยจะอาศัยข้อมูลช่วงปี ค.ศ. 2019 - 2025 ซึ่งสะท้อนถึงราคาที่พักช่วงโควิดและหลังโควิด อีกทั้งยังครอบคลุมถึงเทรนราคาที่พักในปัจจุบัน
คำถามในโปรเจ็คประกอบไปด้วย :
- What pricing trends look like?
- Which neighborhoods and hosts are most active?
- What review activity says about demand?
- Recommendation
